Rechadder

A command line interface chatting application. All the basic features, none of the bloat.

Lightweight

Without bloat like WinForms. ReChadder can host a server and a client, quickly and efficiently. Running fully in the terminal.

Scriptable

ReChadder as intuitive, and advanced scripting for both the server and client using Lua.

Free & Open Source

ReChadder's source is hosted on GitHub. You can choose to build it yourself or use a precompiled binary.
